# 中国传统婴儿取名小程序

基于中国传统文化的智能取名微信小程序，结合传统命理、现代语言学、社会发展等多个维度，为用户提供科学、合理的取名建议。

## 功能特点

- 智能取名推荐
- 多维度名字分析（传统文化、语言学、社会适应等）
- 个性化偏好设置
- 名字评分与解析
- 动态配置管理

## 技术架构

- 前端：微信小程序原生框架（WXML + WXSS + JS）
- 组件库：WeUI
- 状态管理：Mobx-miniprogram
- 云服务：微信云开发
  - 云数据库
  - 云函数
  - 云存储

## 项目结构

```
├── cloudfunctions/    # 云函数
├── miniprogram/      # 小程序源码
│   ├── components/   # 自定义组件
│   ├── pages/        # 页面
│   ├── utils/        # 工具函数
│   └── app.js        # 入口文件
├── docs/             # 文档
└── README.md         # 项目说明
```

## 开发环境配置

1. 安装微信开发者工具
2. 克隆项目
```bash
git clone https://github.com/WenXiangWu/child_name.git
```
3. 在微信开发者工具中导入项目
4. 开启云开发环境

## 本地开发

1. 进入项目目录
```bash
cd child_name
```

2. 安装依赖
```bash
npm install
```

3. 在微信开发者工具中编译项目

## 部署说明

1. 上传云函数
2. 部署云数据库集合
3. 配置云存储权限
4. 发布小程序
