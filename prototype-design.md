# 起名助手小程序原型设计

## 整体架构

小程序采用五个主要Tab页面组成：
- 首页
- 找名
- 找字
- 查诗
- 我的

## 页面设计

### 1. 首页

首页作为小程序的入口，主要包含以下功能区域：
- 起名功能入口：智能分析，多维度评分
- 找字功能入口：多维度查找合适的字
- 快速起名区域：提供姓氏和性别选择，快速生成名字
- 最近起名展示：展示用户最近生成的名字及评分

### 2. 找名页面

找名页面用于搜索和筛选名字，包含以下功能：
- 搜索框：输入名字关键词或寓意
- 筛选条件：
  - 名字风格（传统、现代、文艺、独特）
  - 性别（男、女、通用）
  - 名字长度（单字、双字）
- 搜索结果展示：
  - 名字
  - 评分
  - 性别
  - 寓意
  - 风格标签

### 3. 找字页面

找字页面用于按条件搜索汉字，包含以下功能：
- 搜索框：输入汉字或拼音
- 筛选条件：
  - 部首
  - 笔画
  - 五行属性
  - 字义类别
- 字符列表展示：
  - 汉字
  - 拼音
  - 简要释义
  - 五行属性

### 4. 查诗页面

查诗页面用于搜索和浏览古诗词，包含以下功能：
- 搜索框：输入诗词名、作者或内容
- 筛选标签页：
  - 朝代（全部、先秦、两汉、魏晋、南北朝、隋代、唐代、五代、宋代、金朝、元代、明代、清代、近代）
  - 类别（全部、诗、词、曲、赋、文言文）
  - 主题（山水、田园、边塞、爱情、离别、思乡、送别、抒情、咏物、写景）
- 诗词列表展示：
  - 标题
  - 朝代
  - 作者
  - 内容预览
  - 主题标签

### 5. 我的页面

我的页面作为用户个人中心，包含以下功能：
- 用户信息展示
- 功能入口：
  - 我的收藏
  - 历史记录
  - 意见反馈
  - 设置
- 其他辅助功能入口

## 详细页面

### 名字详情页

展示名字的详细信息和分析：
- 名字展示
- 整体评分
- 多维度分析（音律、字形、寓意、五行等）
- 操作按钮（收藏、分享）

### 字符详情页

展示汉字的详细信息：
- 汉字展示
- 拼音和读音
- 详细释义
- 字形分析
- 五行属性
- 相关名字推荐

### 诗词详情页

展示诗词的详细内容：
- 标题和作者信息
- 完整内容
- 注释功能
- 相关诗词推荐
- 操作按钮（收藏、分享、复制）

## 流程设计

### 起名流程

1. 用户输入基本信息（姓氏、性别）
2. 选择性输入出生信息（出生年月日时）
3. 选择起名偏好（风格、字数、禁用字等）
4. 系统生成名字列表
5. 用户查看名字详情和分析
6. 用户可收藏或分享喜欢的名字

### 找字流程

1. 用户输入搜索条件（部首、笔画、五行等）
2. 系统展示符合条件的汉字列表
3. 用户点击查看字符详情
4. 用户可将喜欢的字加入收藏

### 查诗流程

1. 用户输入搜索条件（关键词、朝代、类别、主题等）
2. 系统展示符合条件的诗词列表
3. 用户点击查看诗词详情
4. 用户可查看注释、收藏或分享诗词

## 技术实现

小程序采用微信原生开发框架，主要技术点包括：
- 页面布局和样式设计
- 表单验证和数据处理
- 本地数据缓存
- 用户信息授权
- 分享功能实现
