
# 🎯 起名系统数据文件使用分析与替换建议报告

## 📊 一、当前数据文件使用现状

### 🔍 **起名生成流程中的数据文件依赖**

#### **1. 核心起名引擎 (`QimingNameGenerator`)**

**数据流程**:
```
用户输入 → 数据加载 → 笔画组合 → 五行匹配 → 常用字过滤 → 标准字符验证 → 三才五格计算 → 结果输出
```

**使用的数据文件**:

1. **五行字典文件**
   - `wuxing_dict_jianti.json` (7027行)
   - `wuxing_dict_fanti.json`  
   - **用途**: 根据笔画数和五行属性获取候选汉字
   - **调用位置**: `data-loader.ts:48, 103`

2. **新华字典预处理文件**
   - `processed/xinhua-processed.json` (142613行)
   - **用途**: 获取汉字的拼音、声调、笔画、部首、五行等基础信息
   - **调用位置**: `data-loader.ts:155`

3. **拼音数据文件**
   - `processed/pinyin-processed.json` (39557行)
   - **用途**: 汉字声调查询的备选数据源
   - **调用位置**: `data-loader.ts:195`

4. **常用字文件**
   - `processed/common-chars-male.json` (6745行)
   - `processed/common-chars-female.json` (5501行)
   - **用途**: 性别化常用名字用字过滤
   - **调用位置**: `data-loader.ts:438`

5. **三才五格规则**
   - `sancai-rules.json`
   - **用途**: 三才五格吉凶判断规则
   - **调用位置**: `data-loader.ts:497`

6. **标准字符验证**
   - `standard-characters.json`
   - **用途**: 通用规范汉字表验证，确保生成的名字符合国家标准
   - **调用位置**: `standard-characters-validator.ts:44`

#### **2. 诗词起名引擎 (`PoetryNamer`)**

**使用的数据文件**:
- `poetry/` 目录下的各类诗词文件 (8个JSON文件)
- `chinese-poetry/` 目录下的古典文学作品 (1000+个JSON文件)
- **用途**: 诗词典籍中提取优美的名字组合

#### **3. 字符信息查询系统**

**使用的数据文件**:
- `word-chunks/` 分片数据 (17个JSON文件)
- `word-index.json`
- **用途**: 汉字详细信息查询和高级搜索
