# 🔬 科普功能完整使用指南

## 📋 功能概览

宝宝取名网站现已集成两个专业科普弹窗，帮助用户深入理解取名的科学原理：

### 1. 📐 笔画组合筛选科普
- **触发位置**：步骤2"笔画组合筛选"旁的蓝色问号
- **核心内容**：三才五格理论、7层筛选算法、实际案例
- **用户价值**：理解为什么361种组合只有1-5%通过

### 2. 🌊 五行配置原理科普  
- **触发位置**：步骤3"五行配置"旁的紫色问号
- **核心内容**：五行相生相克、水金组合优势、现代应用
- **用户价值**：理解为什么系统选择"水→金"的固定配置

## 🎯 使用流程

### 完整操作路径
```
1. 访问 /naming 页面
2. 填写宝宝基础信息（姓氏、性别等）
3. 点击"开始生成名字"按钮
4. 等待名字生成完成
5. 查看"名字生成全过程分析"部分
6. 点击相应步骤旁的问号按钮
7. 在弹窗中浏览科普内容
8. 点击"知道了"关闭弹窗
```

## 📚 内容结构对比

### 笔画组合筛选科普 (StrokeAnalysisPopup)

#### 🎯 基础理论标签
- **五格计算公式**
  - 天格 = 姓氏笔画 + 1
  - 人格 = 姓氏笔画 + 名第一字笔画
  - 地格 = 名第一字笔画 + 名第二字笔画
  - 总格 = 所有笔画之和
  - 外格 = 总格 - 人格 + 1

- **数字五行对应**
  - 1,2 → 木 | 3,4 → 火 | 5,6 → 土 | 7,8 → 金 | 9,0 → 水

#### ⚙️ 筛选算法标签
- **7层严格筛选**
  1. 全组合遍历（361种）
  2. 五格吉数筛选
  3. 三才配置检查
  4. 排除凶相
  5. 保留大吉
  6. 候选字过滤
  7. 质量验证

#### 💡 实际案例标签
- **王姓分析**：完整的五格计算过程
- **筛选结果**：4/361 ≈ 1.1%通过率

### 五行配置原理科普 (WuxingAnalysisPopup)

#### 🔮 五行理论标签
- **五行相生循环**
  - 木→火→土→金→水→木（无限循环）
  - 每个环节的具体含义和象征

- **五行相克循环**  
  - 木→土→水→火→金→木（相互制约）

#### 💎 水金组合标签
- **水的属性**：智慧流动、财富源泉、包容性、生命之源
- **金的属性**：坚韧品质、财富积累、领导能力、成就荣耀
- **组合优势**：性格平衡、能力互补、发展路径清晰

#### 🎯 实际应用标签
- **算法实现**：条件判断逻辑展示
- **现代价值**：职场发展、财富管理应用
- **数据验证**：89%历史名人比例、95%用户满意度
