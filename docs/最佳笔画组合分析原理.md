# 最佳笔画组合分析原理

## 核心理论：三才五格姓名学

最佳笔画组合的分析基于传统的**三才五格姓名学**理论，这是一套完整的数理分析系统。

### 1. 五格计算公式

对于一个完整姓名（如：张浩然），五格计算如下：

```
姓名：张(7画) 浩(10画) 然(12画)

天格 = 姓氏笔画 + 1 = 7 + 1 = 8
人格 = 姓氏笔画 + 名第一字笔画 = 7 + 10 = 17  
地格 = 名第一字笔画 + 名第二字笔画 = 10 + 12 = 22
总格 = 所有笔画之和 = 7 + 10 + 12 = 29
外格 = 总格 - 人格 + 1 = 29 - 17 + 1 = 13
```

### 2. 三才五行转换

通过五格数字的**个位数**确定五行属性：

```javascript
// 数字到五行的映射（取个位数）
1,2 → 木  (甲乙木)
3,4 → 火  (丙丁火) 
5,6 → 土  (戊己土)
7,8 → 金  (庚辛金)
9,0 → 水  (壬癸水)
```

**三才组合**：
- 天才：天格五行 (8 → 金)
- 人才：人格五行 (17 → 金) 
- 地才：地格五行 (22 → 木)
- 三才组合：金-金-木
